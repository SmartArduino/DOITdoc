代码块
---------------------------

code單行引用：顯示單行內容,用雙反引號(``)包起來

code多行引用：顯示多行內容，雙冒號(::)下給他一個區塊

`$sudo apt-get install pygame``

::

    $sudo apt-get install pygame
    $sudo apt-get install python
    $sudo apt-get install python-pip
