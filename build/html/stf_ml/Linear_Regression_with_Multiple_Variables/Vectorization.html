Unvectorized implementation

matlab 是从 \(\theta(1)\) 开始，所以是 1:n+1

prediction = 0.0;
for j = 1:n+1,
  prediction = prediction + theta(j) * x(j)
end;

Vectorized implementation

prediction = theta' * x;

把 求 :math:`theta` 向量化

\(\theta - \alpha \xi\)

$:raw-latex:xi `= :raw-latex:frac{1}{m}`:raw-latex:sum{i=1}^M (h{{:raw-latex:`\theta`}}(x{{(i)}})-y{{(i)}}) x {^{(i)}} $
